# Joe Karlsson Twitch Scripts

## Quickstart

### 1. Clone example repository

```sh
git clone https://github.com/JoeKarlsson/mongodb-graphql-demo.git
cd mongodb-graphql-demo
```

### 2. Setup Nightbot creds

Change `nightbot-creds-example.txt` to `nightbot-creds.txt` and `nightbot-tokens-example.txt` to `nightbot-tokens.txt`

Paste your Nightbot in each file

### 3. Run

```sh
osascript setup-twitch.applescript
```
